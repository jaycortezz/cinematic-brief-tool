# Cinematic Brief Tool

AI-powered cinematic content briefs for cannabis creators. Built with Next.js, Anthropic Claude, and OpenAI image generation.

## Setup

1. Clone this repo
2. Install dependencies: `npm install`
3. Run locally: `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000)

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) and import the repo
3. Deploy — no environment variables needed (keys are entered in the UI)

## Usage

1. Enter your **Anthropic API key** (from console.anthropic.com) for brief generation
2. Enter your **OpenAI API key** (from platform.openai.com) for reference frame images
3. Keys are saved in your browser's localStorage — never sent to any server except the respective APIs
4. Select your subject mode, platforms, and niche
5. Optionally upload mood board images for visual inspiration
6. Generate cinematic briefs with shot lists and AI reference frames
